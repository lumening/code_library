#PHPStorm常用快捷键

## Ctrl

```
ctrl + f 搜索框
ctrl + r 替换框
ctrl + g 跳转指定行框
ctrl + e 最近打开过的文件列表
ctrl + w 选中向左向右的最近区块
ctrl + l 光标跳转到行尾，多次点击向下移动
ctrl + b 跳转到常量、变量、函数、类的申明处，在申明处点击可列出该变量使用列表
ctrl + n 列出phpstrom中内置的类库，如Redis类等
ctrl + / 单行注释、解除单行注释
ctrl + m 让当前光标在屏幕居中
ctrl + j 显示phpstorm预定于模板变量
```

## shift 

```
shift + shift  搜索文件(包括非项目文件)
```

## ctrl + shift

```
ctrl + shift + u  选择某一段文字全部转为大(小)写 
ctrl + shift + l  光标跳转到行尾，多次点击向上移动 *[相近命令：ctrl + l]
ctrl + shift + n  查询phpstrom中文件(包括非项目文件)  *[相近命令：ctrl + n]
ctrl + shift + r  高级替换操作 *[相近命令：ctrl + r]
ctrl + shift + v  查看粘贴板的内容
ctrl + shift + s  另存为新文件
```

## Fn

```
F5      另外为新文件
F6      移动到新目录
F10     定位到导航菜单栏
F12     定位到左侧侧边栏
```
